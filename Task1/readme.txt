Task 1

Задание: Реализовать одно экранное андроид приложение - список. Каждый элемент такого списка содержит картинку и название (порядок по желанию). Изображения загружаются асинхронно из сети при прокрутке. Длина списка - 10 - 20 элементов

Комментарии: На данном этапе использовать ListView. Для загрузки изображения из сети использовать, например, библиотеку Picasso или Glide. Пример приложения (Task 1 - Solution Example.mp4) находится в корне текущей папки. Развернуть и отлаживать приложение как на симуляторе, так и на реальном устройстве (при наличии) - в любом случае, на проверке задания установим на реальный девайс.
